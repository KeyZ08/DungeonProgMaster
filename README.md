# DungeonProgMaster
Проект по программированию

Нус, начинаю хоть как-то делать игру.
Игра наподобии "Час Кода", тоесть будет учить программированию в игровой форме.
Игрыть мы будем Охотником за сокровищами, который отправляется в подземелье, полное богатств, головоломок и врагов.

Делать игру на WinForms очень непривычно и неудобно, поэтому, надеюсь, что выйдет хоть что-то хорошее.
